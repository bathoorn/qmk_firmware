# Ronnie's Sol Layout
## Layout

### Qwerty layout
```
   * ,------------------------------------------------.  ,------------------------------------------------.
   * | ESC  |   1  |   2  |   3  |   4  |   5  |   -  |  |   =  |   6  |   7  |   8  |   9  |   0  | BkSp |
   * |------+------+------+------+------+------|------|  |------|------+------+------+------+------+------|
   * | Tab  |   Q  |   W  |   E  |   R  |   T  |   [  |  |   ]  |   Y  |   U  |   I  |   O  |   P  |   \  |
   * |------+------+------+------+------+------|------|  |------|------+------+------+------+------+------|
   * | CAPS | C(A) | O(S) | A(D) | S(F) |   G  |      |  |      |   H  | S(J) | A(K) | O(L) | C(;) |   '  |
   * |------+------+------+------+------+------| OLED |  | OLED |------+------+------+------+------+------|
   * |Shift |   Z  |   X  |   C  |   V  |   B  |      |  |      |   N  |   M  |   ,  |   .  |   /  |Enter |
   * |------+------+------+------+------+------+------|  |------+------+------+------+------+------+------|
   * | Home | PGDN | PGUP |  END | ADJ  | Space| ENC1 |  | ENC2 | Enter|  FN  | Left | Down | Up   |Right |
   * |------+------+------+------+------+------+------|  |------+------+------+------+------+------+------'
   *                                    | Space| DEL  |  | BkSp | Enter|
   *                                    `-------------'  `-------------'
```

C(_) = Hold for Control or tap for _ key
O(_) = Hold for OS or tap for _ key
A(_) = Hold for Alt or tap for _ key
S(_) = Hold for Shift or tap for _ key

## Customize

see `qmk_firmware/keyboards/sol/rev1/keymaps/default/rules.mk`
and `qmk_firmware/keyboards/sol/rev1/keymaps/ronnie/rules.mk`

```

# Variables you can set for SOL

TRILAYER_ENABLED = yes      # Enable the Adjust layer
ENCODER_ENABLE = yes        # Enable encoders
RGB_OLED_MENU = 0           # Set first encoder as OLED menu controller


```
## Compile

go to qmk top directory.
```
$ cd qmk_firmware
```

build
```
$ make rgbkb/sol:ronnie
```

After the initial flash with AVRdudess, you should be able to flash using this:
```
$ make rgbkb/sol:ronnie:dfu
```
